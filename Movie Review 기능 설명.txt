프로젝트명 : Movie Review – (JPA – 데이터베이스 기반)

투입인원 : 1명

프로젝트 소개 :  영화 관련 정보를 업로드 수정 삭제하는 웹 사이트 / JPA 를 실제 사용해보기 위하여 제작

사전 작업 :
1. 데이터베이스 기획 및 구현
2. 필요한 자료 수집

기능 구현
1. 페이지 기능
	페이지버튼이 필요한 화면에다가 삽입
	출력할 게시물의 개수에 비례해서 필요한 페이지 버튼이 생성 
     Ex :  1  or  1 2 3 4 5  
	삼각형을 클릭 시 이전 혹은 다음 페이지로 이동
	숫자를 클릭 시 해당 페이지로 이동

2. 영화 목록 페이지
	보유한 영화 목록 출력
	원하는 영화를 클릭 시 해당 영화 상세페이지로 이동 : 4번 기능
	검색창을 이용해 원하는 호텔을 검색 가능 : 3번 기능

3. 검색 기능
	작성되어진 키워드를 repository을 통해 데이터 베이스 데이터와 비교함으로써 해당되는 게시물만 출력
	SELECT 쿼리 문 사용

4. 영화 상세 페이지
	영화의 기본키를 이용하여 하나의 HTML에 클릭되어진 영화 정보가 넘어와서 출력
	Update, Delete, Back 버튼을 통해서 해당 기능 구현
	Update 클릭 시 : 수정 페이지로 이동 : 5번 기능
	Delete 클릭 시 : 메시지 출력 후 확인 시 데이터 베이스와 목록에서 삭제 
	Back 클릭 시 : 영화 목록 페이지로 이동 :  2번 기능

5. 영화 수정 페이지
	기본키를 활용하여 영화 상세 페이지에 업로드할 정보를 수정하여 재 업로드 및 데이터 베이스에 업데이트
	삭제  /  삽입 / 수정은 Transaction을 활용

개발도구 :

∙ DB설계 - MySQL Developer
∙ IDE 도구 - IntelliJ
∙ 웹 서버 – Spring Boot 
∙ 언어 - JAVA / JavaScript / HTML /CSS
